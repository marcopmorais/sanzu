# 26 — Glossary

| Term | Meaning |
|---|---|
| Case / Processo | One post-loss administrative journey |
| Manager (Process Manager) | Funeral agency operator role |
| Editor | Family member responsible for uploading/execution |
| Reader | View-only family member |
| Step library | Master step definitions (modular) |
| Rules engine | Generates the case plan from inputs |
| Workstream | Parallel track (banks, insurance, benefits…) |
| Evidence | Documents/receipts/confirmations proving progress |
| Evidence graph | Linked proof that defines “done” |
| Assisted submission | Connector action with explicit approval and receipt |
| Rework | Reopened steps or re-uploaded docs due to issues |
| Cycle time | Time from case creation to closure |
| Time-to-next-action | Time from unblock to next meaningful action |
| RBAC | Role-based access control |
