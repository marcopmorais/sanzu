# 18 — GTM, Sales Enablement and Pilot Playbook

## GTM model
**B2B2C via Funeral Agencies**
- Agency is buyer + operator (Manager)
- Family is end user (Editor + Readers)

## Beachhead
- 10–20 pilot agencies with recurring volume and willingness to professionalize admin operations

## Pilot (30–60 days)
### Setup
- Train 1–2 managers per agency
- Define success metrics and baseline estimates
- Run first 5 cases with close support

### Success metrics
- case creation <2 minutes (median)
- end-to-end completion rate (pilot)
- status calls reduced (self-report) ≥30%
- family NPS ≥40; agency NPS ≥40
- rework <10%

## Sales narrative (agency)
1) Families are overwhelmed; agencies absorb pressure via calls and repeated requests  
2) Ceremony is handled; bureaucracy remains chaotic  
3) Sanzu structures the admin process into a guided workflow with documents and shared status  
4) Outcome: fewer calls, less rework, higher professionalism  
5) Offer: pilot with measurable ROI

## Assets to create
- 1-page ROI sheet
- 10-min demo script
- Security & privacy FAQ
- Pilot proposal (2–3 pages)
