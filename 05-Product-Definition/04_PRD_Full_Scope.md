# 04 — PRD (Full Scope, Multi-Version)

## 1) Problem
Families face a fragmented post-loss administrative maze; agencies absorb pressure manually. The system lacks a structured digital execution and transparency layer.

## 2) Goals by phase
| Phase | Goal | Outcome |
|---|---|---|
| V1 | Deterministic clarity + docs | reliable end-to-end completion |
| V2 | Scale agency operations | more cases per manager; fewer calls |
| V3 | Automation with approvals | less manual admin work |
| V4 | Mission control orchestration | unified workstreams with verified completion |

## 4) Key metrics
- Adoption: active agencies/managers
- Engagement: steps completed/case, docs uploaded/case, templates generated
- Efficiency: median cycle time, time-to-next-action
- Quality: missing-doc loops, rework rate
- Satisfaction: NPS family + agency
- Economics: support minutes/case, gross margin
