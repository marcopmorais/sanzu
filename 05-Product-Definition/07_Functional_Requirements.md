# 07 — Functional Requirements (Full)

## V1 (Foundation)
- Auth + invitation onboarding
- Case management (create/edit/archive)
- RBAC enforcement + audit
- Questionnaire (progressive, save/resume, branching)
- Workflow engine (steps, dependencies, statuses)
- Document vault (upload/download, tags, versioning, restricted)
- Template generation (pre-fill + missing-field prompts + PDF)
- Dashboards (progress, next best step, missing docs)
- Notifications (lite: invites + key reminders)
- Audit export

## V2 (Agency scale)
- Multi-case agency dashboard (search, filters, workload)
- SLA indicators (time-to-next-action, blockers)
- Analytics (case funnel, cycle time distribution, rework)

## V3 (Automation + assisted submissions)
- Notification rules engine (configurable, safe defaults)
- External request object model (create → approve → submit → receipt)
- Connector framework + receipt capture
- Status tracking per external party

## V4 (Mission control)
- Workstreams orchestration (parallel execution, dependencies)
- Evidence graph (facts/docs/confirmations)
- Status polling + reconciliation (integrations)
- Evidence export package (closure proof)
